# FairPlay Rail

AI-powered Anti-Bot Engine for IRCTC Tatkal Ticket Bookings.

## Features
- Detects suspicious booking behavior
- Analyzes user interaction patterns
- Predicts bot vs genuine user
- Web-based interface

## Tech Stack
- Python
- Flask
- Machine Learning
- HTML/CSS/JavaScript

## How to Run
1. Clone the repository
2. Install dependencies
3. Run app.py
